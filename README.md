QRStudio is a modern, responsive web application for generating highly customizable QR codes. Create QR codes for websites, Wi-Fi networks, digital menus, resumes, PDFs, text, phone numbers, WhatsApp, and more. Preview them instantly on real-world mockups and export them in high quality.

✨ Features
🎨 Customizable QR codes (colors, patterns, logos, and styles)
🌐 Generate QR codes for:
URLs / Websites
Wi-Fi Networks
Digital Menus
Resume / Portfolio Pages
PDFs & Files
Text Messages
Phone Numbers
WhatsApp Links
👀 Live preview with real-world mockups
🌙 Dark & Light mode support
📱 Fully responsive design
💾 Download and export QR codes instantly
🚀 Tech Stack
HTML5
CSS3
JavaScript
QR Code Generation Library
Responsive UI Design
🎯 Goal

QRStudio aims to make QR code creation simple, beautiful, and accessible for everyone—from students and professionals to businesses and restaurants.
